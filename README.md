# CheckDigit

Ruby implementation of the Luhn, Damm and Verhoeff check digit algorithms

## Installation

Add this line to your application's Gemfile:

    gem 'check_digit'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install check_digit
